# Contributing to CashClaw Sprint1000

Thanks for helping improve a practical system that aims to help ordinary people earn their first $1,000 with ethical AI-assisted services.

## What to contribute

- Better outreach and closing templates
- Market-specific playbooks with measurable outcomes
- Improvements to model routing and cost/performance balance
- Bug fixes for API stability and deployment experience

## Contribution flow

1. Fork the repository
2. Create a branch: `feat/your-topic` or `fix/your-topic`
3. Make focused changes with clear rationale
4. Add or update documentation
5. Open a pull request using the PR template

## Quality expectations

- Keep changes small and testable
- Document assumptions and risks
- Include before/after behavior
- Avoid over-promising outcomes

## Reporting strategy results

When sharing growth feedback, include:
- Market and channel
- Offer used
- Outreach volume
- Response rate
- Close rate
- Revenue generated

Evidence-driven reports help us improve faster.
